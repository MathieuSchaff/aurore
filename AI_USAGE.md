# AI Usage

I used AI tools (ChatGPT / Claude) during this project mainly as a helper, not as a replacement.

## What I used it for

- Generating boilerplate (routes, schemas)
- Refactoring some functions
- Finding edge cases for tests
- Debugging specific issues

## What I did myself

- Architecture decisions (project structure, stack choices)
- Reviewing and adapting all generated code
- Writing and adjusting business logic
- Testing and validating features manually

## Important

I never copy-pasted code without understanding it.  
If something wasn’t clear, I rewrote or simplified it.

---

AI helped me go faster, but I made sure to stay in control of the code.
